# Testing Strategy

This project uses unit tests and integration tests.

Unit tests check individual functions like add(), subtract(), multiply(), and divide().

Integration tests verify the interaction between input and the calculation logic.

## Testing Pyramid

Most tests are unit tests, which follow the testing pyramid principle.

## Black-box testing

Tests focus on checking outputs for given inputs without looking inside the implementation.

## Regression testing

Running the test suite after changes ensures that existing functionality continues to work.
